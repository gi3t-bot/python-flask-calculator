# Python Flask Calculator

A web-based calculator application built using Flask.  
This project demonstrates core Python backend concepts along with basic web integration using Flask, HTML templates, and static assets.

## Features

- Basic arithmetic operations
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Clean and simple user interface
- Server-side calculation using Flask
- Input validation for safe operations

## Tech Stack

- Backend: Python, Flask
- Frontend: HTML, CSS
- Template Engine: Jinja2
- Version Control: Git and GitHub

## Installation and Setup

1. Clone the repository

git clone https://github.com/gi3t-bot/python-flask-calculator.git  
cd python-flask-calculator

2. Create and activate a virtual environment

python -m venv venv  
venv\Scripts\activate        (Windows)  
source venv/bin/activate     (Linux or Mac)

3. Install dependencies

pip install -r requirements.txt

4. Run the application

python app.py

5. Open in browser

http://127.0.0.1:5000/
